# Homes

> 집을 구하는 사회초년생을 위해 서울에 있는 아파트, 오피스텔, 단독 주택, 연립 주택 등 다양한 주택의 매매, 계약 정보와 주변 정보를 지도에서 확인하고 검색해 볼 수 있는 웹 애플리케이션
<br>
